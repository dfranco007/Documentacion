\cdpchapter{Resumen}

\thispagestyle{empty}

A lo largo de este Proyecto de Fin de Carrera se ha realizado una paralelización mediante OpenMP de una aproximación al algoritmo de segmentación de imágenes \textit{level set}. El fin de esta segmentación es la extracción de ciertas características de las imágenes necesarias para el cliente con el que se trabaja a lo largo de este proyecto. 

Además, se ha realizado un estudio previo de los tipos de técnicas de segmentación existentes y se han realizado pruebas experimentales con la paralelización realizada, de manera que se pudiera comprobar la mejora realizada frente al algoritmo en serie. 

\

\textbf{Palabras clave}: paralelización, OpenMP, \textit{level set}, segmentación, imagen.
